1.2、Git常用命令

1.2.1、将代码提交到本地仓库

  • 首先进入到工作目录下
  • 进行初始化：git init—–》生成一个隐藏的目录
  • 查看状态：git status 红色则为没添加到仓库
  • 添加进入仓库：git add *
  • 提交文件 ： git commit -m ‘注释’
  • 如果对代码进行修改 那么查看状态 就会有爆红的地方，显示有修改
  • 再将文件添加进入仓库：git add *
  • 然后进行提交：commit

1.2.2、将代码上传到远程仓库

  • 生成密钥 ssh-keygen -t rsa -C “1730765741@qq.com“ 一路默认 就会将密钥生成再本地

  • 将密钥添加进入牛客网

  • //在本地声明这个远程仓库，名字为origin
git remote add origin https://git.nowcoder.com/355668118/gitTest01.git

//只能传到master分支
